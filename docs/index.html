01 · Como funciona

Do pedido na conversa até o texto pronto, em 5 etapas

Cada etapa abaixo existe no código. O servidor roda na sua máquina e conversa com o Claude pelo protocolo MCP (stdio) — nada passa por servidor de terceiros além das fontes que cada tool usa.

1

Pedido no Claude

Você pede em linguagem natural ("resume esse vídeo", "o que aparece na tela?") e o Claude escolhe a tool certa do servidor.

→
2

Identificação do vídeo

O servidor extrai o ID de qualquer formato: watch?v=, youtu.be, shorts, embed ou o ID direto de 11 caracteres.

→
3

Rota de extração

A tool escolhida define a rota: legendas via yt-dlp, metadados via youtubei.js, vídeo para o Gemini ou áudio para o Whisper local.

→
4

Processamento

Parse das legendas sem duplicatas com fallback de idioma (pt → en → automática), espera do processamento no Gemini ou transcrição local.

→
5

Resposta e limpeza

O resultado volta em Markdown estruturado (título, idioma, duração, palavras) e todos os arquivos temporários são apagados.

A etapa 3 se ramifica em quatro rotas — uma por tool:

Legendas (yt-dlp)

Baixa só as legendas no formato json3, o mais limpo — texto sem duplicatas, timestamps opcionais.

Metadados (youtubei.js)

Título, canal, duração, visualizações e a lista de legendas disponíveis — sem autenticação.

Visão (Gemini)

Baixa o vídeo em baixa qualidade, envia para a File API do Google e o Gemini analisa o que aparece na tela.

Áudio (Whisper local)

Baixa só o áudio e transcreve na sua máquina — funciona até para vídeo sem nenhuma legenda.

02 · As 4 tools

Quatro maneiras de ler um vídeo — cada uma com seu custo

Três das quatro tools funcionam sem chave de API nenhuma. Só a análise visual usa a API do Gemini, com chave gratuita criada no Google AI Studio.

youtube_transcript

Extrai a transcrição pelas legendas do vídeo (manuais ou automáticas) usando yt-dlp. Pede em português por padrão e cai para inglês automaticamente se não houver. Aceita timestamps opcionais em cada trecho.

yt-dlp sem API key

youtube_info

Retorna título, canal, duração, visualizações, descrição e a lista completa de legendas disponíveis (manuais e automáticas) via youtubei.js. É a tool de reconhecimento: mostra o que as outras têm para trabalhar.

nenhuma dependência extra sem API key

youtube_vision

Análise visual de verdade: baixa o vídeo em baixa qualidade, envia para a File API do Google e o Gemini responde sobre o que aparece na tela — slides, código, interfaces, cenas — e não só sobre o áudio.

yt-dlp GEMINI_API_KEY (Google AI Studio)

youtube_whisper

Baixa só o áudio e transcreve localmente com o OpenAI Whisper — a saída para vídeo sem legenda nenhuma. Modelos de tiny (rápido) a large (preciso); tudo roda na sua máquina.

yt-dlp openai-whisper (pip) FFmpeg 100% local, sem API key

03 · Instalação guiada

Cole um prompt no Claude Code e ele conduz tudo

O README traz um prompt pronto de onboarding: o Claude verifica os pré-requisitos, instala o que faltar, cria a chave do Gemini no Google AI Studio por você — controlando o navegador, com você fazendo apenas o login — registra o servidor e valida cada tool com um vídeo real antes de dizer que terminou.

1

Clone o repositório

Só precisa de Node.js 18+ e do Claude Code instalados. O resto o onboarding resolve.

2

Cole o prompt do README

Um único prompt, em bloco de código, pronto para copiar — está na seção de instalação do README.

3

O Claude conduz o resto

A cada etapa de navegador (como criar a chave do Gemini), ele pergunta com botões se você quer que ele faça — via Playwright ou Claude in Chrome — ou te guia no passo a passo manual. Ao final, testa tudo com um vídeo público.

# 1. clonar e entrar na pasta
git clone https://github.com/ericlucianoferreira/youtube-mcp.git
cd youtube-mcp

# 2. abrir o Claude Code na pasta
claude

# 3. colar o prompt de onboarding do README
#    e responder aos botões — o Claude faz o resto
Sobre a chave do Gemini: ela só é necessária para a tool de análise visual (youtube_vision). As outras três tools funcionam sem chave nenhuma — e a chave, quando você quiser, é criada em minutos no Google AI Studio, sem cartão de crédito. A chave fica só na sua máquina (.env / registro do MCP), nunca na conversa.

04 · Stack

Um arquivo, poucas dependências, tudo local

O servidor inteiro vive em um único index.js. As dependências de sistema (yt-dlp, Whisper, FFmpeg) são opcionais conforme as tools que você quiser usar.
